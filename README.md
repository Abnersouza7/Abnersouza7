## 🚀 Featured Projects

### [DevKit-Zephyr](https://github.com/Abnersouza7/DevKit-Zephyr) — Zephyr RTOS Dashboard
Desktop tool for embedded engineers working with **Zephyr**: real-time log visualization, system metrics (heap / stack / threads), and debugging insights — all without rebuilding with debug configs.

### [vTaskSoS](https://github.com/Abnersouza7/TrabalhoFinalSTR) — Real-Time Scheduling on FreeRTOS
Final project for *Real-Time Systems (EMB-5633, UFSC)*. Implements **Rate Monotonic** scheduling, **Priority Inheritance Protocol**, and a **Polling Server** for aperiodic tasks on Arduino UNO + FreeRTOS.

### [Bellevue Lighting Control](https://github.com/Abnersouza7/BellevueControlPage) — WebSocket-driven LED Dashboard
Production interface for [Evolution Virtual](https://lightitwithvibrant.com)'s VIBRANT API. Real-time channel control of LED lighting boards over WebSocket, with passcode auth and responsive UI.

### [Processamento Digital de Sinais](https://github.com/Abnersouza7/Processamento-Digital-de-Sinais)
DSP coursework — filter design, transform analysis, and signal reconstruction in Python.
